# taskly

An API to manage tasks with authentication.

## Technology Stack
- node.js
- mongodb
- mongoose
- typescript
- koa
- jest
- jwt

## Running the project

Install the dependencies:
```
npm i
```

Serve:
```
npm start
```

## Running the tests
```
npm test
```

## Linting
```
npm run lint
```

## Compiling
```
npm run compile
```

## Docker

```
docker-compose up
```
